# mcpal

A CLI for the [Model Context Protocol](https://modelcontextprotocol.io). `aws` for MCP — discover servers configured by your other clients (Claude, Cursor, opencode, Zed, …), connect to local stdio or remote Streamable HTTP, send any request.

Status: early. Building toward M1 (stdio + read surface).
